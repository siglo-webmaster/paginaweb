Resumen

Misión y Alcance

¿Qué problema resuelve este proyecto?
Resuelve limitaciones de la pagina web actual en cuanto a:
1. Usabilidad: permitirá tener una pagina web mas fácil de usar e intuitiva para el cliente.
2. Escalabilidad: Permitirá tener un sistema modular, fácilmente actualizable y modificable
3. Automatización: El sistema permitirá tener tareas programadas para su actualización y programación de eventos
4. Tecnología: La nueva pagina estará desarrollada en base a tecnologías actuales que son soportadas ampliamente por los proveedores de hosting.
¿Cuál es el objetivo de este proyecto?
Elaborar la nueva pagina web, que sera una herramienta importante en la toma de contacto y venta
¿Cuál es el alcance de este proyecto?
El proyecto ira hasta la elaboración de la pagina web, sincronizar esta con el sistema Siglo y tener las interfaces de administración pertinentes para su uso cotidiano.

Estado

24-julio-2012 : En el momento la pagina web cuenta con carro de compras, maquina de búsqueda, área de cliente, integración con plataforma de pagos, área de administración con los módulos de  destacados y eventos. Esta cargando el catalogo de siglo, tirant y publidisa, sincroniza con el inventario y catalogo proveniente de Siglo.

Se requiere hacer labores de depuración y pruebas del aplicativo.
